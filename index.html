Q6. Print the Pattern.Write a program which ask user for no of lines and print a pattern using an asterik .

A6. const Print_pattern = (N) =>
{
for(var i=1; i<=N; i++)
{
for(var j=1; j<=i; j++)
{
process.stdout.write("*");
}
process.stdout.write("\n");
}
};
